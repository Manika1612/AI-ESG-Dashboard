# AI ESG Compliance Dashboard

This project is an AI-powered ESG analytics tool built using real BRSR (Business Responsibility and Sustainability Reporting) data.

It helps identify:

* Which companies are ESG compliant
* Which sectors are underperforming
* Where disclosure risks exist

The goal is to simulate how investors, analysts, and regulators evaluate ESG performance using data and AI.

**Features**

* ESG compliance scoring
* Company & sector comparison
* Principle-wise insights
* Risk identification
* AI-generated insights (simulated)

**Tech Stack**

* Python
* Pandas
* Streamlit

**Use Case**

Helps analyze ESG performance and identify compliance gaps across companies.


## 🚀 What Makes This Interesting

* Built on real-world ESG reporting data (not dummy data)
* Converts qualitative disclosures into measurable scores
* Enables multi-level analysis (company, sector, principle)
* Designed as a decision-support tool, not just a dashboard
